Frequently Asked Questions

How do I start a game?

For a round to start there must be at least one player on each team.

Why can't I become (INSERT PROP HERE)

If a prop has a red outline, either you do not have enough room to become that prop, or you need to wait for the cooldown timer to expire. If there is no outline then that prop is not allowed to be chosen.
